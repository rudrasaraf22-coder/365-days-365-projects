# Day 13 — Dice Roll Simulator

Folder: `day 13 Dice Roll simulator`

Simple web-based dice roll simulator built for the "365 days 365 projects" series.

Credits

- Built and credited to: @rudrasaraf22-coder (https://github.com/rudrasaraf22-coder)

How to use
1. Open `index.html` in your browser.
2. Click "Roll Dice" to roll once.
3. Click "Roll 10×" to run 10 rapid rolls for a quick demo.

Files
- index.html — UI
- style.css — Styling
- script.js — Dice logic (uses Unicode dice faces)
- README.md — This file

Notes
- No build tools required — plain HTML/CSS/JS.
- If you'd like an image-based die, autoplay on load, or additional features (history, statistics), tell me and I can add them.
