# Day 12 — Age Calculator

Simple Age Calculator project for "365 days — 365 projects".

What it does
- Accepts a Date of Birth (DOB) using an HTML date input.
- Calculates the age in years, months, and days compared to the current date.
- Validates empty input and future dates and displays appropriate messages.

Files
- `index.html` — main UI and wiring.
- `style.css` — styles for layout and presentation.
- `script.js` — age calculation logic and DOM interactions.

How to use
1. Open `index.html` in a browser.
2. Pick or type your date of birth.
3. Click "Calculate" (or press Enter) to see your age displayed as years, months, and days.
4. Click "Clear" to reset.
